Clocks

A fully featured and rendered 3D Clock using WebGL.

View on GitHub Download .zip Download .tar.gz

Features Include:

  • Starting & Stopping the hands
  • Setting your own time
  • Automatically retreiving the correct time from the Internet
  • Increasing/Decreasing hand speeds
  • Zoom In/Out
  • Alarm
